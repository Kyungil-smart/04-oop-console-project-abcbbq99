# 콘솔 프로잭트 개요
- 만들어 볼 게임 : 다키스트던전 1 스타일의 탐험 RPG
- 제목 : 미정

- 조작 
  - 메뉴
    - 상하 : 커서 이동
    - 선택 : 엔터
  - 키보드 상하좌우 = 이동
  - 인벤토리 : I
    - 인벤토리 커서 이동 : 상하
    - 선택 : 엔터
  - 종료 : Esc

- 구현 목표 : 거래 시스템 / 장비 시스템 / 전투 시스템
